# password_manager
This is a Python-based Password Vault uses tkinter and JSON to create a secure, visual tool for generating, saving, and searching your login credentials. It streamlines your security by offering one-click password generation, clipboard copying via pyperclip, and a search feature to instantly retrieve account details from a local database.

Save all files including the photo in same folder 

run pip install json
run pip install pyperclip
